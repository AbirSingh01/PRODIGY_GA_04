Task 4 - Image-to-Image Translation with cGAN

Objective

Implement an image-to-image translation model using a conditional Generative Adversarial Network (cGAN) based approach to transform an input image into a modified output image.

Technologies Used

- Python
- PyTorch
- Diffusers
- Hugging Face
- PIL (Python Imaging Library)

Files

- "task4.py" - Main Python script
- "input.jpg" - Input image
- "output.png" - Generated output image
- "output_gateway.png" - Additional generated output
- "README.md" - Project documentation

Process

1. Load a pre-trained image-to-image translation model.
2. Provide an input image.
3. Apply a text instruction to modify the image.
4. Generate and save the translated image.

Results

Successfully generated translated images from the input image using a pre-trained generative AI model.
